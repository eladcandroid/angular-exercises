⭐ Storybook - תרגילי בונוס מתקדמים

אתגרים מתקדמים להרחבת הידע ב-Storybook

💡 סקירה

סיימתם את התרגילים הבסיסיים? מעולה! עכשיו הגיע הזמן להתקדם לנושאים מתקדמים יותר של Storybook. תרגילי הבונוס יעזרו לכם להפוך את התיעוד לאינטראקטיבי יותר, נגיש יותר, ומקצועי יותר.

🎭 בונוס 1: Play Functions מתקדמות

מתקדם

צרו play function מורכבת שמדגימה תרחיש שלם של משתמש בספרייה:

  • סינון ספרים לפי קטגוריה
  • חיפוש ספר ספציפי
  • לחיצה על ספר
  • מילוי טופס השאלה
  • אימות שהטופס נשלח בהצלחה

דרישות:

  • השתמשו ב-within, userEvent, ו-expect
  • בדקו מצבי ביניים (לדוגמה, טעינה)
  • אמתו שכל ה-actions נקראו
  • טפלו בזמני המתנה (waitFor)

💡 רמז:

השתמשו ב-canvas.findByText לאלמנטים שמופיעים אסינכרונית, וב-waitFor לאימות שינויי מצב. ראו דוגמאות ב-demo/components/loan-form/loan-form.stories.ts.

♿ בונוס 2: בדיקות נגישות מקיפות

בינוני

צרו stories שבודקות נגישות מלאה של כל הקומפוננטות:

  • ניווט מקלדת (Tab, Enter, Escape)
  • ARIA labels ו-roles
  • ניגודיות צבעים
  • Focus management
  • תמיכה ב-screen readers

דרישות:

  • השתמשו ב-@storybook/addon-a11y
  • הגדירו parameters.a11y לכל story
  • צרו play function שבודקת keyboard navigation
  • וודאו שאין אזהרות נגישות

📱 בונוס 3: Responsive Stories

בינוני

צרו stories שמציגות את הקומפוננטות במסכים שונים:

  • Desktop (1920x1080)
  • Tablet (768x1024)
  • Mobile (375x667)
  • Custom breakpoints

דרישות:

  • השתמשו ב-parameters.viewport
  • הגדירו custom viewports אם צריך
  • צרו story אחד לכל גודל מסך
  • וודאו שהעיצוב responsive

🧩 בונוס 4: Component Composition

מתקדם

צרו story מורכב שמשלב את כל הקומפוננטות למערכת שלמה ופועלת:

  • סינון אינטראקטיבי
  • הצגת תוצאות מסוננות
  • פתיחת modal עם טופס
  • עדכון סטטיסטיקות

דרישות:

  • השתמשו ב-render function
  • צרו signals ו-computed לניהול state
  • חברו בין כל הקומפוננטות
  • הוסיפו אינטראקציות מלאות

🌙 בונוס 5: Dark Mode

בינוני

הוסיפו תמיכה ב-Dark Mode לכל הקומפוננטות:

  • צרו CSS classes ל-dark mode
  • השתמשו ב-parameters.backgrounds
  • צרו story לכל theme
  • וודאו ניגודיות טובה

🎨 בונוס 6: Custom Decorators

קל

צרו decorators מותאמים אישית:

  • RTL layout decorator
  • Padding/margin decorator
  • Background color decorator
  • Border decorator
